First-time setup (one-time, takes ~5 min)

To enable the visual editor, a site admin needs to:

  1. Sign up for a free Netlify account (no credit card needed).
  2. Connect this wiki's GitHub repository to Netlify — it'll deploy a mirror of the site.
  3. On the Netlify site, enable Identity (Site settings → Identity → Enable).
  4. Under Identity → Services, enable Git Gateway.
  5. Invite yourself (and co-editors) as Identity users.

After that, anyone you invite can click Log in here, edit articles and reviews in a visual editor, and their changes commit to GitHub automatically.

Full instructions: Decap CMS documentation.

Quick alternative: edit on GitHub

You don't have to wait for the CMS. Go to any page on EndoWiki, scroll to the bottom, click ✏️ Edit this page on GitHub. GitHub has a web editor; you just need a free GitHub account.
